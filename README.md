# Projeto 1 – Árvores e Balanceamento

Implementação das seguintes estruturas de dados:

- Árvore Binária de Busca (BST)
- Árvore AVL
- Árvore Rubro-Negra

Operações implementadas:

- Inserção
- Remoção
- Busca
- Cálculo de altura

Também foi implementada uma heurística aproximada para o Problema do Caixeiro Viajante utilizando o algoritmo **Nearest Neighbor**.

## Experimentos

Foram realizados experimentos com três tamanhos de entrada:

- 10 cidades
- 50 cidades
- 100 cidades

Cada experimento foi executado **30 vezes** e foi calculado o **tempo médio de execução**.
